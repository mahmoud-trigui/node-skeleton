# 🔨 node-project-template

A simple repo template to quickly bootstrap a new node project

## How to use

💻 run in dev watch mode:

```
npm run dev
```

🚀 run in production mode:

```
npm run start
```

🔬 run tests:

```
npm run test
```

🔬 run tests in watch mode:

```
npm run test-watch
```

🚿 clean up build folder:

```
npm run clean
```

🏗 build project:

```
npm run build
```
